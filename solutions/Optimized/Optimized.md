This optimized program:
Reads a text grid and direction sequence from files.
Rotates each grid row left or right depending on the directions.
Stores the transformed rows.
Extracts the middle row.
Computes the ASCII sum of its characters.
Outputs and saves the result
The code is optimized by:
Using lines() for fast reading,
Clean rotation logic,
Reduced string operations,
Efficient character looping,
Automatic resource handling.
